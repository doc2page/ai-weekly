# AI Weekly

> 每周5分钟，看懂AI圈发生了什么

## 关于

AI Weekly 是一份面向普通人的AI周报，用口语化方式解读本周AI领域最重要的进展。

### 特点

- 🗣️ **社区在吵什么** - Hacker News、Reddit上的热门话题
- 🛠️ **普通人能用什么** - 新工具、新产品、新玩法  
- 💡 **这跟你有什么关系** - 每条内容都有实用价值
- 🔮 **接下来会发生什么** - 基于热点的趋势预测
